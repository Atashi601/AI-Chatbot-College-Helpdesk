# 🤖 AI Chatbot (College Helpdesk)

## 📌 Project Overview
The AI Chatbot (College Helpdesk) is a simple rule-based chatbot built using Python. It helps students by answering frequently asked questions related to college such as admission, fees, courses, placement, hostel, and facilities.

This project demonstrates a basic AI conversational system using keyword-based logic.

---

## 🎯 Objectives
- To build a simple chatbot for college enquiry system  
- To provide instant responses to student queries  
- To reduce manual interaction in helpdesk services  
- To demonstrate basic rule-based AI/NLP concept  

---

## 🛠️ Technologies Used
- Python (Core Programming Language)  
- Rule-based Logic (If-Else Keyword Matching)  
- VS Code / Any Python IDE  

---

## 💡 Features
- Admission information  
- Fee structure details  
- Courses offered  
- Placement details  
- Hostel facilities  
- Library information  
- Instant response system  
- Command-line chat interface  

---

## ⚙️ System Workflow
User Input → Keyword Detection → Condition Matching → Response Output
